# web-editor

## How To Run

Make sure you have Node and NPM installed on your workspace.
  - node (v20.10.0)
  - npm (v10.2.3)
  - Basic knowledge of TypeScript
  - Basic knowledge of React or Angular


**Build**
```
npm run build

```
**Run**

```
npm run dev

```
*NOTE*: Make Sure that the Websocket service is running
