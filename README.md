# How to develop your own WebEditor with Monaco and Language Server

## Prerequisites

### Language Server
  - Java (v17)
  - IDE of your choice that supports Java

### Web application
  - node (v20.10.0)
  - npm (v10.2.3)
  - Basic knowledge of TypeScript
  - Basic knowledge of React or Angular
